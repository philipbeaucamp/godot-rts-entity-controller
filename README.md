# Godot RTS Entity Controller

A Godot addon containing building blocks for creating RTS games by providing entity (i.e. units, buildings etc) controlling logic, such as selecting, moving and attacking (other) entities.

Includes a sample unit and ready to play scene.

# Documentation

For detailed documentation please visit the [official documentation](https://philipbeaucamp.github.io/godot-rts-entity-controller/) or use [mkdocs](https://www.mkdocs.org/) to build the site locally.

# Demos

An additional playable demo and reference project with extra content (including additional units and abilities) can be found at [itch.io](https://philipbeaucamp.itch.io/godot-rts-entity-controller)
