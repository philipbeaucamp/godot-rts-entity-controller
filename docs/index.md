# Welcome to the Godot RTS Entity Controller Docs

To download an executable demo or purchase the full godot project visit [todo](https://philipbeaucamp.itch.io/)



## Overview

* `mkdocs new [dir-name]` - Create a new project.
* `mkdocs serve` - Start the live-reloading docs server.
* `mkdocs build` - Build the documentation site.
* `mkdocs -h` - Print help message and exit.


## Downloadable content

Both the godot addon (which includes one minimally configured unit) as well as a executable demo can be downloaded for **free** to try out the tool and get a feel for it. An advanced reference project with multiple (todo how many) units, different types of unit abilities and attack behaviours and more level setups can be purchased separately.

- Game demo [FREE]: Executable that demonstrate the gameplay and feel of the Godot RTS Entity Controoler
- Addon [FREE]: Contains all core scripts, one basic unit and a few assets required to use the tool
- Reference project [PAID]: Contains multiple levels and units demonstrating different unit behaviour and capabilities of the tool, as well as a few cosmetic assets and shaders.
